# Bouncing Ball Animation

Анимация на C# Windows Forms с движущейся картинкой, которая меняет направление при достижении границ и переключает изображение в зависимости от направления движения.

<img width="883" height="394" alt="bouncing-ball-animation" src="https://github.com/user-attachments/assets/5df07b0e-2e4b-495c-b07a-77b4bc1c882c" />

## Функциональность

- Движение картинки слева направо и обратно
- Изменение изображения при смене направления:
  - Движение вправо → картинка "Ris2.png"
  - Движение влево → картинка "Ris1.png"
- Отскок от границ формы (левая и правая границы)
- Управление скоростью через TrackBar
- Кнопки пуск/стоп

## Интерфейс

- pictureBox1 — движущаяся картинка
- timer1 — таймер анимации (обновление позиции)
- trackBar1 — регулировка скорости
- button1 — запуск анимации
- button2 — остановка анимации
- button3 — выход из программы

## Пример работы

При запуске анимации картинка движется:
- Вправо с изображением Ris2.png
- При касании правой границы (640px) меняет направление налево и изображение на Ris1.png
- При касании левой границы (0px) снова меняет направление направо и изображение на Ris2.png
- Скорость регулируется ползунком

## Куда можно встроить

- В игровые проекты
- В демонстрационные приложения
- В обучающие программы по анимации
- В интерактивные презентации

## Требования

- Windows
- .NET Framework / .NET Core с WinForms
- Файлы изображений Ris1.png и Ris2.png в папке с программой

## Примечание

Изображения должны находиться в той же папке, что и исполняемый файл. При остановке анимации движение прекращается, при повторном запуске продолжается с того же места.
